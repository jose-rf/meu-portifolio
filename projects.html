Jogo do Número Secreto

O Jogo do Número Secreto é uma aplicação interativa desenvolvida como um projeto de aprendizado. Este jogo simples desafia os jogadores a adivinharem um número secreto escolhido aleatoriamente dentro de um intervalo definido. O jogador tem a oportunidade de fazer uma série de palpites, recebendo dicas sobre se o número secreto é maior ou menor que o palpite atual.
